Step to Run project :
step1 : create a locacl db 
step2: change config in .env for db 
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=<dbnae>
DB_USERNAME=root
DB_PASSWORD=<password>
step3 : migrate db using this command =>  php artisan migrate:fresh
step4 : Run the project using this command  => php artisan serve
